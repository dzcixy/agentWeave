# Data directories

- `traces/`: raw or converted AgentWeaver JSONL traces.
- `profiles/`: measured vLLM/SGLang profiling CSVs and fitted latency models.
- `processed/`: context segment graphs, DAG exports, and aggregate events.
- `results/`: CSV outputs from baselines and wafer replay.
- `plots/`: generated PDF/PNG figures.

Large traces and model artifacts are intentionally not committed.
